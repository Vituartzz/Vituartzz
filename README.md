### Featured Project

#### [Albion Market Pro](https://github.com/Vituartzz/albion-dev)

Real-time market analysis tool for **Albion Online** — scans prices across all cities, identifies arbitrage opportunities, and calculates profit after taxes.

**Stack:** Next.js 15 · React 19 · TypeScript · Prisma · PostgreSQL · Redis · Tailwind CSS · Zod · TanStack Table

**Highlights:**
- Arbitrage engine with confidence scoring algorithm
- Real-time price normalization from Albion Data Project API
- Profit calculator with premium/non-premium tax rules
- Subscription-based access control (Scout / Trader / Whale)
- Redis caching with stale-while-revalidate fallback
